# eSCAPE-API


API documentation for [**eSCAPE**](https://escape-model.github.io/)

+ Local build:

```bash
cd eSCAPE
python setup.py install
cd doc 
make html
```

+ Commit to Github Page

Copy the `doc/_build/html` repository to `eSCAPE-API.github.io`, then commit.
